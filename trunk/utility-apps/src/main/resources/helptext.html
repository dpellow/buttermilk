cryptoregistry.com Registration Key Materials® Wizard

Introduction

Welcome to the registration wizard. cryptoregistry.com uses a simple JSON-formatted file format for transactions. This wizard helps build up the data file required to request an initial registration. Later stages of the registration (such as payment for crypto services or requesting Identity Points®) are handled online. The initial registration process including using this software, upload, and submission is free.

The registration wizard consists of several tabs. These are intended to be used more or less in order moving from left to right:

  1. Introduction (this tab)
  2. Select Registration Handle
  3. Initial Key
  4. Personal Contact
  5. Business Contact
  6. Signature

Each tab builds the registration file starting with the registration handle itself and then moving through your initial encryption key, contact info appropriate to your situation, and finally a digital signature using your own key.

FAQ

What Is cryptoregistry.com?

cryptoregistry.com is part of a new Public Key Infrastructure (PKI) designed by David R. Smith. The PKI design consists of an open source cryptographic library implemented in the Java programming language, called Buttermilk; a new security protocol (loosely modeled on TLS) called bTLS (aka "beatles"); new cryptographic formats, utilities, and programs, and a web site to support it all.
